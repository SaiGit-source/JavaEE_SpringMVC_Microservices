Maven Project Update:

By default Maven will use Java 1.5 version or Java 5 version. 

JRE System Library [J2SE-1.5]

if you want to change, go to pom.xml -> whichever Java you like to work with -> add here -> <maven.compiler.source>14</maven.compiler.source> that's Java 14

Right click pom.xml file -> Maven -> Update Project

then folder changes to J2SE-14
